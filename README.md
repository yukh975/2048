# 2048

Браузерная игра 2048 — один HTML-файл, без зависимостей.

## Возможности

- **4 уровня сложности:** Сложно (4×4), Средне (5×5), Просто (6×6), Легко (8×8)
- **Таймер** с паузой — отсчёт времени от начала игры
- **Управление:** стрелки на клавиатуре, свайпы (мобилка), перетаскивание мышью
- **Отмена хода:** кнопка ↶ или Ctrl+Z / Cmd+Z (один раз за ход)
- **Сохранение:** очки, рекорд, таймер и состояние поля сохраняются в localStorage
- **Умные свайпы:** пустой свайп (ничего не сдвинулось) тихо игнорируется
- **Анимации:** появление и слияние плиток, всплывающие очки
- **Справка:** кнопка `?` с описанием управления и правил
- **Favicon** в формате SVG

## Запуск

Открыть `index.html` в браузере. Всё.

## Управление

| Действие | Способ |
|---|---|
| Сдвинуть плитки | ← ↑ ↓ → / свайп / мышь |
| Отменить ход | Ctrl+Z / Cmd+Z / кнопка ↶ |
| Новая игра | Enter / кнопка «Новая игра» |
| Пауза | P / кнопка ⏸ |
| Справка | Кнопка `?` |
