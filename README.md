# 2048

Браузерная игра 2048 — один HTML-файл, без зависимостей.

## Возможности

- **3 уровня сложности:** Легко (3×3), Классика (4×4), Сложно (5×5)
- **Управление:** стрелки на клавиатуре, свайпы (мобилка), перетаскивание мышью
- **Отмена хода:** кнопка ↶ или Ctrl+Z / Cmd+Z (один раз за ход)
- **Сохранение:** очки, рекорд и состояние поля сохраняются в localStorage
- **Анимации:** появление и слияние плиток, всплывающие очки
- **Справка:** кнопка `?` с описанием управления и правил

## Запуск

Открыть `index.html` в браузере. Всё.

## Управление

| Действие | Способ |
|---|---|
| Сдвинуть плитки | ← ↑ ↓ → / свайп / мышь |
| Отменить ход | Ctrl+Z / Cmd+Z / кнопка ↶ |
| Новая игра | Кнопка «Новая игра» |
| Справка | Кнопка `?` |
